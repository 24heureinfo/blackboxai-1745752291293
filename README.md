# Ma Boutique

## Project Overview
Ma Boutique is an online shopping website that showcases a collection of products for sale, including sports shoes, elegant watches, backpacks, and sunglasses. The website is designed to provide an aesthetically pleasing user interface while ensuring a smooth shopping experience. Users can view products, check prices, and add items to their shopping cart.

## Installation
To run the Ma Boutique project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ma-boutique.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ma-boutique
   ```
3. Open the `index.html` file in your preferred web browser to view the application.

## Usage
Once you have opened the `index.html` file in a web browser, you can explore the available products. Each product displays its name, image, and price, along with an "Add to Cart" button that allows users to include the item in their shopping cart.

## Features
- Responsive design using Tailwind CSS for a modern and clean look.
- Display of multiple products with images, names, and prices.
- Functional navigation bar for easy access to different sections of the website.
- Cart icon that shows the number of items added to the cart.

## Dependencies
This project utilizes the following external libraries:
- **Tailwind CSS**: A utility-first CSS framework used for styling the page.
- **Font Awesome**: Provides icons used in the navigation and cart interface.
- **Google Fonts (Roboto)**: Used for the typography of the website.

## Project Structure
The project consists of the following file:
- `index.html`: The main HTML file that contains the structure of the webpage and links to external stylesheets and scripts.

### File Description
- `index.html`: Contains the HTML structure, including the header, main content area with products, and footer.

Feel free to modify the content and style as needed to fit your specific requirements or preferences.
